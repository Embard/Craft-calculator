# Bezzumie Craft

Веб-калькулятор крафта в атмосфере Зоны.

## Что уже есть
- STALKER-inspired оформление
- добавление нескольких предметов в список крафта
- сводка только по нужным ресурсам
- поддержка изображений у предметов и ингредиентов

## Как добавить реальные картинки
В `data.js` каждому предмету или ингредиенту можно добавить поле:

```js
image: 'assets/items/container-2.png'
```

Если картинки нет, показывается стилизованный placeholder с короткой меткой.
